📈 AI-Powered Market Intelligence Terminal

A modular AI-powered financial market monitoring platform built with Python, Streamlit, automated scheduling systems, and local LLM integrations.

Overview

This project was developed to monitor financial markets, macroeconomic conditions, Treasury yields, volatility, software-sector performance, and market-moving news in real time.

The system combines live financial APIs, AI-generated macro summaries, automated market alerts, charting systems, and scheduled email briefings into a single modular platform.

Core Features

  • 📊 Real-time market dashboard
  • 📈 Interactive market charts
  • 🤖 AI-generated macroeconomic market briefs
  • 🚨 Intraday market alert system
  • 📧 Automated daily email summaries
  • 📰 Financial news aggregation
  • 💾 Daily market brief archive
  • ⚡ Modular Python backend architecture
